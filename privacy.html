Privacy Policy

Last updated: August 31, 2025

1. Information We Process

  • Extension settings (tone, length, license status).
  • Optional profile data imported from Upwork/Fiverr (stored locally in your browser via chrome.storage).
  • Anonymous usage counters for free quota (stored locally).

2. Data Sources

The extension reads job page content in your browser to generate a tailored draft. We do not transmit this data to our servers unless you explicitly enable a cloud model requiring an API key.

3. Third Parties

  • Model providers (e.g., OpenAI) — used only if you opt-in and provide an API key.
  • Payment providers (e.g., Paddle, PayPal) — handle billing and payouts.

4. Security

We use browser storage and HTTPS endpoints where applicable. You are responsible for safeguarding your license key and any model API keys.

5. Your Rights

You may delete the extension data at any time by removing the extension or clearing browser storage. For billing data, contact the payment provider.
